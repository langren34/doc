AnythingLLM 测试文档

这是一个用于演示文档上传的示例文档。

主要内容：
- 第一章：项目概述
- 第二章：技术架构
- 第三章：使用说明

第一章 · 项目概述
AnythingLLM 是一个全能 AI 应用，由 Mintplex Labs 开发。
它集成了 RAG、AI Agents、可视化工作流等多种功能。

第二章 · 技术架构
- 前端：Electron + React
- 后端：Node.js + Express
- 向量数据库：LanceDB（默认）/ Chroma / Pinecone 等
- LLM Provider：OpenAI / Anthropic / Ollama 等

第三章 · 使用说明
1. 安装 AnythingLLM 桌面版
2. 配置 LLM Provider（推荐 DeepSeek）
3. 创建工作区
4. 上传文档
5. 开始对话
